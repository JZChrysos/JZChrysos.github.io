Wang Tile Game

The "Domino Problem," introduced by Hao Wang in 1961, asks whether a given set of square tiles with their edges colored can tile the plane (without rotation) such that neighboring tiles' edges match. Berger showed in 1966 that this was an undecidable problem, implying the existence of tilesets which can only tile the plane aperiodically. Berger provided such a tileset containing over 20,000 tiles. Over the years the size of the smallest known aperiodic Wang tile set came down and down, and finally in 2015 Jeandel and Rao showed that the smallest such tile set has exactly 11 tiles and 4 colors. This little interactive game allows you to explore the 11-tile set they discovered. To learn about the math, check out Jeandel and Vanier.
